Front-end Curated

Curated tools and resources for front-end developers

Star

Bootstrap

Bootstrap makes front-end web development faster and easier. It's made for folks of all skill levels, devices of all shapes, and projects of all sizes.

Foundation

Foundation is a family of responsive front-end frameworks that make it easy to design beautiful responsive websites, apps and emails that look amazing on any device.

Skeleton

Light as a feather at ~400 lines & built with mobile in mind. Styles designed to be a starting point, not a UI framework. Quick to start with zero compiling or installing necessary.

Tachyons

Create fast loading, highly readable, and 100% responsive interfaces with as little css as possible. The entire library is fewer than 14kb when minified and gzipped.

Basscss

Basscss is a lightweight collection of immutable utilities designed for speed, clarity, performance, and scalability. Using clear, humanized naming conventions, Basscss is quick to internalize and easy to reason about while speeding up development time with more scalable, more readable code.

Pure

Pure is ridiculously tiny. The entire set of modules clocks in at 4.0KB* minified and gzipped. Crafted with mobile devices in mind, it was important to us to keep our file sizes small, and every line of CSS was carefully considered.

Neat

Neat is a fluid grid framework with the aim of being easy enough to use out of the box and flexible enough to customize down the road.

Jeet

Jeet allows you to express your page grid the same way a human would describe it. No more needlessly nesting elements. No more rigid twelve column rules.

Semantic UI

Semantic is a UI component framework based around useful principles from natural language. Semantic is a development framework that helps create beautiful, responsive layouts using human-friendly HTML.

Unsemantic

Unsemantic is a fluid grid system that is the successor to the 960 Grid System. It works in a similar way, but instead of being a set number of columns, it's entirely based on percentages.

Susy

Susy is an agnostic set of tools for creating powerful, custom layouts. We provide the language, but you provide all the opinions. Use a grid, don't use a grid, or use a combination of grids â€” it's all up to you.

Materialize

A modern responsive front-end framework based on Material Design.

Toast

The Toast framework is a grid. That's it. Soon, it might have type styles and whatnot, but its power is in its highly customisable design.

UIkit

A lightweight and modular front-end framework for developing fast and powerful web interfaces. UIkit gives you a comprehensive collection of HTML, CSS, and JS components which is simple to use, easy to customize and extendable.
